USER GUIDE

How Does Someone Use Vocation By Location?

Vocation By Location is a REST Service.
The user will need to send an HTTP GET requests to use the service. It does not matter what language they use to send this request.
These requests will be secure (https).
The responses from this service will come back as JSON Objects.
No Authentication of any kind is needed.

In order to use the service, go to our site located here: https://ourcompany.com
Once there, you will be able to send requests and receive responses to those requests.

You will need to send at least two requests through our service to get the desired results.

You Send:
A session will require two HTTP GET requests:
1. An HTTP GET request looking for the industry.
2. An HTTP GET request looking for the locations.

You Receive:
A session will return a list of JSON objects for each request.
1. A JSON Object that contains the industry name and its corresponding id.
JSON Object for Industry: {industry name, id}.
2. A JSON Object that contains the location name and its corresponding state, and population.
JSON Object for Location: {location name, state, population}.
