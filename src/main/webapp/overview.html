OVERVIEW

What Is Vocation By Location?

Vocation by Location provides a geographical concentration data of industries in the US.
It is a service designed to help the user find an industry, and then use that indusrty to find locations in the US where that industry is thriving based on population.


How Does Vocation By Location Work?

We provide a URI to process these requests.
There are two endpoints for the requests made through our service.

  1. Industries
  2. Places
A user sends a request to receive a list of industries. Each industry result that we provide will have an industry name and an id associated with it.
The user will then take the id that they have and pass this into the URI from our site.
This will get them a list of locations that feature the highest concentrations for that industry in the US.


Who Is Vocation By Location Designed For?

This service is geared towards anyone that would need this kind of data for an application, or a project or even a job search.
You just need our site and to have a working knoweldge of how to send an Http request.
