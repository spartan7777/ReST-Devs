OVERVIEW

What Is Vocation By Location?

Vocation by Location provides a geographical concentration data of industries in the US.
It is a service designed to help the user find an industry, and then use that industry to find locations in the US where that industry is thriving based on population.


How Does Vocation By Location Work?

Our service does three things:

  1. Reaches out to a Service to get the requested data.
  2. Processes this data to make it more user friendly.
  3. Brings this back to the user so that they can utilize it.

We provide a URI to process these requests.
There are two paths for the requests made through our service.

  • Industries: http://13.58.130.252:8080/ReST_Devs/api/industries

    • Or

  • Places: http://13.58.130.252:8080/ReST_Devs/api/places?industry={Some industry id}

A user sends a request to receive a list of industries. Each industry result that we provide will have an industry name and an id associated with it.
The user will then take the id that they have and pass this into the Places URI from our site as a query parameter. (Please refer to our User Guide for more info on supported query parameters.)
This will get them a list of locations that feature that industry in the US.


Who Is Vocation By Location Designed For?

This service is geared towards anyone that would need this kind of data for an application, or a project, or even a job search.
You just need our site and to have a working knowledge of how to send an Http request.
