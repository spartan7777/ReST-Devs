What Do Results From Vocation By Location Look Like?

This page contains examples of what responses to successful requests from Vocation By Location looks like.
There are examples of requests for both industries, and the corrpesonding requests for the locations for those industries.

The Example below displays the results for a successful industry request.

Example 1: A successful response to a request for the "Computer Systems Design" industry.

A. Name: The name for the industry that was requested.
B. Average Wage: The average wage for the industry that was requested.
C. Industry ID: The corresponding id for the industry that was requested, in this case 5415.


The Examples below display the results for a successful corresponding place request.
These responses provide the places for the "Computer Systems Design" industry, represented by the industry id 5415.

Example 2: A successful response to a place request using the provided industry id.

A. Name: The name for the place where the requested industry is located.
B. State: The state where the place is located for the selected industry.
C. Population: The corresponding population for the place where the industry is located.


Example 3: Another successful response to a place request using the provided industry id.

A. Name: The name for the place where the requested industry is located.
B. State: The state where the place is located for the selected industry.
C. Population: The corresponding population for the place where the industry is located.


Example 4: An additional successful response to a place request using the provided industry id.

A. Name: The name for the place where the requested industry is located.
B. State: The state where the place is located for the selected industry.
C. Population: The corresponding population for the place where the industry is located.
