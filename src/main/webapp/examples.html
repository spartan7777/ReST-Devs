What Do Results From Vocation By Location Look Like?

This page contains examples of what responses to successful requests from Vocation By Location looks like.
There are examples of requests for both industries, and the corrpesonding requests for the locations for those industries.

The Example below displays the results for a successful industry request.

Example 1: A sample of a successful response to an industry request.

http://13.58.130.252:8080/ReST_Devs/api/industries



A. The name for the industry.
B. The average wage for the industry.
C. The corresponding id for the industry, in this case 5415. This is what's used to get place data.


The Examples below display the results for a successful corresponding place request.
These responses provide the places for the "Computer Systems Design" industry, represented by the industry id 5415.

Example 2: A sample from a successful response to a place request using the provided industry id.

http://13.58.130.252:8080/ReST_Devs/api/places?industry=5415



A. The name for the place where the requested industry is located.
B. Companies per thousand people
C. The state where the place is located for the selected industry.
D. Number of companies
E. The corresponding population for the place where the industry is located.
